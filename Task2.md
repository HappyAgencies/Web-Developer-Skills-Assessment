# Task 2: Style the HTML Page with CSS

## Objective

Add CSS styles to the HTML page created in Task 1 to improve its appearance.

## Requirements

- Create a CSS file named `styles.css`.
- Link the CSS file to the HTML file.
- Add styles for the heading, paragraph, and image.

## Instructions

1. Create a new file named `styles.css` in the root directory of your project.
2. Link the `styles.css` file to your `index.html` file.
3. Add CSS rules to style the heading, paragraph, and image.

## Detailed Explanation

### Step-by-Step Guide

1. **Creating the CSS File**:
   - Create a file named `styles.css`.

2. **Linking the CSS File**:
   - Add a `<link>` element inside the `<head>` of `index.html` to link the CSS file.

3. **Adding CSS Styles**:
   - Add styles for the heading (e.g., font size, color).
   - Add styles for the paragraph (e.g., font size, line height).
   - Add styles for the image (e.g., width, border).

## FAQs

### 1. How do I link a CSS file to an HTML file?

You can link a CSS file to an HTML file using the `<link>` element inside the `<head>` of the HTML file, with the `rel` attribute set to `stylesheet` and the `href` attribute pointing to the CSS file.

### 2. How do I change the font size and color of a heading in CSS?

You can change the font size and color of a heading using the `font-size` and `color` properties in CSS.

### 3. How do I style an image in CSS?

You can style an image using CSS properties like `width`, `height`, `border`, etc. Use the `img` selector to target the image element.

Once you have completed Task 2, commit the `styles.css` file to your forked repository and update your `index.html` file to link to the CSS file. Include a note in your README file explaining how to view the styled HTML page.
